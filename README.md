# MarkingCode Assistant

**MarkingCode Assistant** — это локальное приложение для генерации, хранения и управления кодами маркировки (включает DataMatrix, EAN-13 и EAN-8), а также работы с внешне сохранёнными кодами (добавленными руками в БД).  
Предназначено для тестирования, отладки и быстрого ввода данных в системы, поддерживающие эмуляцию сканера через JavaScript или ADB, а так же генерацию визуальных кодов.

---

## 🔧 Зачем это нужно?

- **Генерация кодов маркировки** по заданным шаблонам (например, для ЧЗ)
- **Хранение каталога кодов** с возможностью быстрого поиска и повторного использования
- **Эмуляция сканирования** через копирование JS-вызова (для браузерных консолей)
- **Работа с внешними кодами**, полученными из других источников (файлы, API, сканеры)
- **Быстрая подстановка** ранее сохранённых записей обратно в генератор

---

## ⚙️ Требования

- Права администратора для редактирования локальной базы данных
- Интернет (требуется доступ к ресурсу https://barcode.tec-it.com/)
- Windows 10\11 (вероятно работать будет и на старших версиях, но не проверялось)

---

## 🚀 Быстрый старт

1. Установите через файл mc_assistant_setup.exe
2. Запустить от имени администратора

## 📁 Структура интерфейса

Приложение разделено на три вкладки:
1. Генератор
2. Внешние коды
3. Каталог

---

### 🏷️ Вкладка «Генератор»

Создание новых кодов маркировки на основе шаблонов.

#### Поля:

- **Тип кода** — выбор между `DataMatrix`, `EAN-13`, `EAN-8`
- **EAN** — обязательное поле (13 или 8 цифр). Проверка валидности реализована на стороне barcode.tec-it.
- **Группа маркировки** — отображается только для `DataMatrix`; берётся из `marking_groups.json`
- **Название номенклатуры** — для удобства хранения в каталоге
- **Комментарий** — необязательное поле для заполнения, хранит дополнительную информацию

#### Кнопки:

- **«Сгенерировать код»** — строит полный код и отображает его + изображение DataMatrix/EAN
- **«Сохранить изображение»** — ❌ _временно недоступна_
- **«Скопировать код вызова»** — копирует JavaScript-команду для эмуляции сканирования в браузере
- **«Сохранить в БД»** — сохраняет запись в локальный справочник
- **«Скопировать ADB-команду»** — ❌ _временно недоступна_

> 💡 После генерации все кнопки активируются

---

### 📥 Вкладка «Внешние коды»

Добавление и управление кодами, полученными не из приложения генератора (например, из песочницы).

#### Поля:

- **Название** — обязательно для заполнения
- **Полный код маркировки** — многострочное поле для вставки внешнего кода
- **Комментарий** — необязательное поле для заполнения, хранит любую текстовую информацию

#### Кнопки:

- **Сохранить внешний код** — добавляет запись в каталог с типом Внешний код (Save)
- **Скопировать GS** — копирует символ группового разделителя `\x1D` (если вставленный код не содержит разделитель)
- **Сгенерировать 4/6/8 символов** — генерируют и копируют в буфер случайные строки указанной длины (латинские буквы + цифры)

---

### 📚 Вкладка «Каталог»

Просмотр, поиск и управление всеми сохранёнными записями.

#### Фильтры:

- **Тип записи** — «Все», «Сгенерированные», «Внешние»
- **Группа маркировки** — применяется только к сгенерированным записям
- **Поиск по названию** — живой поиск при вводе

#### Действия над записью (справа):

- **↑ (Зелёная стрелка)** — подставить данные в «Генератор» (только для сгенерированных записей)
- **👁️ (Глаз)** — предпросмотр изображения кода (только для внешних записей)
- **📋 (Буфер обмена)** — скопировать JS-вызов для эмуляции сканирования
- **🗑️ (Корзина)** — удалить запись из Каталога

> 💡 Таблица автоматически обновляется при входе на вкладку

---

## 📂 Файлы конфигурации

- `marking_groups.json` — шаблоны для генерации DataMatrix (поддерживает `{random:N}`, `<GS>`)
- `db/database.db` — локальная база данных (создаётся автоматически)

---

## 🛠️ Известные ограничения

- Редактирование групп маркировки реализовано посредством сохранения формата в marking_groups.json (в файле **mc_groups_redactor.md** отдельно описано как его редактировать)
- Кнопки **«Сохранить изображение»** и **«Скопировать ADB-команду»** в табе «Генератор» временно не реализованы.
- Предпросмотр работает только для внешних кодов; для сгенерированных — используйте переход в «Генератор».
