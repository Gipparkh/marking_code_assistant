# Руководство по редактированию групп маркировки

Файл `marking_groups.json` содержит **шаблоны (маски)** для генерации кодов DataMatrix в соответствии с требованиями систем цифровой маркировки.

---

## 🔧 Синтаксис шаблонов

В масках поддерживаются следующие **специальные теги**:

|Тег|Описание|Пример|
|---|---|---|
|`{ean}`|Подставляется EAN-13, указанный пользователем|`0104601234567890...`|
|`{random:N}`|Генерирует **N случайных символов** (латинские буквы + цифры)|`{random:6}` → `A3K9X2`|
|`{gs}`|Заменяется на символ **группового разделителя** `\x1D` (GS)|Используется как разделитель между блоками данных|

> 💡 Все остальные символы в маске остаются без изменений.

---

## ⚠️ Ограничения

1. **EAN должен быть 13-значным** — приложение не проверяет контрольную сумму.
2. **Случайные строки** генерируются из набора:  
    `ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789`
3. Символ `{gs}` **обязательно** должен использоваться там, где требуется `\x1D` — прямая вставка `\x1D` в JSON **не сработает**.
4. Маска применяется **только для DataMatrix** — для EAN-13/EAN-8 используется только поле EAN.

---

## 📄 Структура файла `marking_groups.json`

Файл представляет собой **JSON-объект**, где:

- **Ключ** — название группы (отображается в выпадающем списке)
- **Значение** — строка-маска

Пример:

{
  "Пиво": "010{ean}215{random:6}{gs}93{random:4}"
}

---

## 🏭 Группы по умолчанию

В поставке приложения используются следующие шаблоны:

|Группа|Маска|Описание|
|---|---|---|
|**Пиво**|`010{ean}215{random:6}{gs}93{random:4}`|Стандартный код для пивной продукции|
|**Пиво БА**|`010{ean}215{random:6}{gs}93{random:4}`|То же, что и «Пиво» (дублирующая группа)|
|**Вода**|`010{ean}215{random:12}{gs}93{random:4}`|Удлинённый криптохвост (12 символов)|
|**Напитки БА**|`010{ean}215{random:12}{gs}93{random:4}`|Аналогично «Воде»|
|**Молочная продукция**|`010{ean}215{random:5}{gs}93{random:4}`|Короткий криптохвост (5 символов)|
|**Молочная продукция с весом**|`010{ean}215{random:5}{gs}93{random:4}{gs}310{random:6}`|Дополнительный блок веса (`310...`)|

> 🔍 Префиксы `01`, `21`, `93`, `310` соответствуют стандартам GS1:
> 
> - `01` — GTIN (EAN)
> - `21` — серийный номер
> - `93` — дополнительный идентификатор
> - `310` — вес в килограммах (6 цифр)
>
  *эта информация добыта нейронкой, достоверность требуется подтвердить*

---

## ✏️ Как добавить или изменить группу

1. Откройте файл `marking_groups.json` в любом текстовом редакторе.
2. Добавьте новую строку в формате:
    "Название группы": "ваша_маска_здесь"
3. Сохраните файл.
4. Перезапустите приложение — изменения вступят в силу автоматически.

> 💡 Убедитесь, что JSON остаётся **валидным** (запятые, кавычки, фигурные скобки).

---

## 🛠️ Пример: создание новой группы

Хотите добавить группу для **табака** с маской `01{ean}21{random:8}{gs}91{random:5}`?

Добавьте в файл:

json:

`{`
  `...,`
  `"Табак": "01{ean}21{random:8}{gs}91{random:5}"`
`}`

После перезапуска она появится в списке «Группа маркировки».
